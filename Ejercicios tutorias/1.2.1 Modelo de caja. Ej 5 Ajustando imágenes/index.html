Ej 5: Ajustando imágenes

Primer caso: padding y borde y box-sizing: content-box

Con el box-sizing: content-box (que es el valor por defecto, como si no pusieramos nada), con las propiedades width y height estamos indicando el tamaño del contenido de ese elemento. Lo que ocupa el padding o el border no forma parte del tamaño indicado en width o height y el elemento ocupará más de lo indicado.

Segundo caso: padding y borde y box-sizing: border-box

Cuando se usa el valor box-sizing: border-box, el tamaño que especificamos en width o height es el tamaño total del elemento. Y en esa medida está incluido el contenido, el padding y el borde de ese elemento.
